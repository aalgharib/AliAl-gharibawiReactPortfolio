# Ali Al-gharibawi — Portfolio

Personal portfolio for Ali Al-gharibawi, AI Software Engineer and founder of [Synosys](https://synosys.io). Built with React, Vite, Tailwind CSS, and Framer Motion.

**Live:** https://alialgharibawi.vercel.app/

---

## Tech Stack

- **Framework:** React 18 + Vite
- **Styling:** Tailwind CSS (Space Grotesk font)
- **Animations:** Framer Motion
- **Icons:** React Icons
- **Deployment:** Vercel

## Sections

- **Hero** — Introduction with CTA buttons
- **About** — Professional background and links
- **Tech Stack** — Animated technology icons (React, Python, Node.js, Next.js, TypeScript, Tailwind, Flask, MongoDB, PostgreSQL)
- **Experience** — Work history at Synosys, Al-Mahdi Centre, Windsor Renovation to Perfection
- **Projects** — 6 featured projects in a card grid (Voice Agents, Synosys Platform, AI Doctor, Multilingual Chatbot)
- **Contact** — Email and phone with direct mail CTA

## Getting Started

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```

## SEO

- Full meta tags, Open Graph, and Twitter Card in `index.html`
- JSON-LD Person schema for AEO/rich results
- `public/sitemap.xml` and `public/robots.txt` included
- Google Search Console verification tag included
